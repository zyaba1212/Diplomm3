# Z96A - Network Architecture Project

## Описание
Дипломный проект "Архитектура системы обработки транзакций для инфокоммуникационных систем в условиях частичного отсутствия сетевого соединения".

## Технологии
- Django 4.2
- Python 3.11+
- Three.js для 3D визуализации
- Solana Web3 для интеграции с блокчейном
- SQLite (для разработки)
- HTML5/CSS3/JavaScript
